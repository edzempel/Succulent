# Succulent
Class assingment for ICS 499 - Software Engineering Capstone

## Instructions

### Database
1. Install XAMPP
2. Run MySQL server on port 3306
3. Open Database GUI
4. Run database configuration script

### Cake
1. Install composer on IDE machine
2. Install cake
